Scheme Parser

Category : Functional Programming
Client : Dalhousie
Completion : February 2019
Role : Coder

A scanner for scheme--, which outputs a token stream. The type of token, and its location is outputed to the screen here.

Created using: Chez Scheme Version 9.5.1 To run enter: "scheme --program keithScanner.ss keithScanner.ss ". To run, make sure the MIT Scheme compiler is installed and simply execute the code from the github repo: https://github.com/keithmacinnis/schemeScanner
